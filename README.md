# Project1-belgianBeers
Ce site a pour but de présenter un top 10 des bières belges, ainsi que les lieux où les déguster à Lyon.
Il est accessible ici : https://emmanuelleallain.github.io/belgianBeers/page1accueil.html

Il s'agit du premier projet réalisé en groupe de 4 personnes à la Wild Code School.

Objectif : 
Réaliser un site statique en 2 semaines en mobile first

Contraintes : 
- Utiliser uniquement du CSS natif
- Intégrer des animations via Javascript
- Avoir un formulaire de contact
